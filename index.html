⚙️ Configuration requise

Avant d'utiliser ce mini-site, tu dois créer une App Registration Azure AD. Cela prend ~5 minutes. Fais-le une seule fois.

Étape 1 — Créer l'App Registration Azure

  1. Va sur portal.azure.com → Microsoft Entra ID → App registrations → New registration
  2. Nom : Klint Academy | Type : Single-page application (SPA)
  3. Redirect URI : l'URL de ce fichier HTML (ou http://localhost pour tester)
  4. Note le Application (client) ID et le Directory (tenant) ID

Étape 2 — Accorder les permissions

  1. Dans l'App Registration → API permissions → Add a permission → Microsoft Graph
  2. Delegated : User.Read + Sites.ReadWrite.All
  3. Cliquer Grant admin consent

Étape 3 — Configurer ce fichier

  1. Ouvre ce fichier HTML dans un éditeur de texte
  2. Cherche VOTRE_CLIENT_ID et remplace par ton Application (client) ID
  3. Sauvegarde et rouvre le fichier — c'est bon !

Étape 4 — Créer la liste SharePoint

  1. Va sur le site SharePoint AnthropicAcademy
  2. Paramètres → Ajouter une application → Liste personnalisée
  3. Nom : SuiviFormation (exactement)
  4. La liste sera automatiquement configurée au premier login
